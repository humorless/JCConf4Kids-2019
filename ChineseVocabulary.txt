------- Programming concepts -------

symbol 符號/標示子

function 函數 (用來包住某一區塊的程式碼，使該區塊自成一塊新的邏輯結構)

parameters 參數 (位於函數的方括弧裡的變數，通常用於從「函數的外界」夾帶資訊進入「函數內部」)

invocation of function 啟動函數

Compile 編譯

compile error 編譯期錯誤

runtime 執行
